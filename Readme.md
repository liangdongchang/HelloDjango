### 创建虚拟环境

```
python -m venv my_venv
```

激活虚拟环境：

```
cd my_venv
cd Scripts
activate
```



### 导入库

```
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```



### 运行

运行项目后，在浏览器输入：http://127.0.0.1:8000/wolf/ 即可访问。
